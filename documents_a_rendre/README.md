Fonctionnalites principales du projet:

- chargement d'etat YAML (`GameState`),
- ramassage d'une clé,
- ouverture d'une porte avec verification de clé,
- transition vers une autre map YAML,
- système de vie avec 3 cœurs affichés en haut à droite,
- interaction avec le `Bandit` qui retire un cœur,
- game over à 0 coeur avec suppression de `state.yml`.
