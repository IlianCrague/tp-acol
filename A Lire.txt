gradle/* utilisé pour installer gradle

src/main/* : le code source du projet
src/test/* : les tests du projet

build.gradle.kts : fichier de configuration gradle
settings.gradle.kts : options du projet gradle

gradlew : launcher gradle linux
gradlew.bat : launcher gradle windows

tp-acol-1.0-SNAPSHOT-all.jar : Fichier jar compilé du projet


Compilation :
Linux / MacOS

./gradlew shadowJar

Windows
.\gradlew.bat shadowJar

Execution : 
Pour lancer la version qui vient d'être compilée
java -jar build/libs/tp-acol-1.0-SNAPSHOT-all.jar

Pour lancer la version à la racine :
java -jar tp-acol-1.0-SNAPSHOT-all.jar
